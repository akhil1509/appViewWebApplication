C n Shoot




COST : 40/-
To register for the event contact Help desk at GRIET.
Spot registrations are available.
JUDGING CRITERIA : Based on no. of words, answers in the least time
DESCRIPTION
 ROUND-1:
Technical word building. Participants will be provided a bowl of coloured pebbles so that one pebble should be chosen. According to it a technical word is given to him/her and with the last letter of the word, he should build a technical word. The chain goes on. He will be given a limited time for it. The shortlisted participants will be qualified to next round.
ROUND-2:
Archery shooting. We provide an Archery board with the concepts in ‘C’ like functions, strings, pointers etc. Participants have to shoot an arrow at the board to select a topic. Concepts will be asked based on the topic he aimed. Participants who answer them correctly in least time will be shortlisted to next round.
ROUND-3:
A simple code in C is given with some errors in it, the participants have to identify the mistakes and make Necessary corrections for the compilation. The participant also have to write the code in simple way. One who does it in a best way will be declared as winner .
