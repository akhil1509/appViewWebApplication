JUDGING CRITERIA : Based on correctly accomplished tasks
DESCRIPTION
 ROUND-1:
The participants are supposed to pick a question stuck randomly on the benches .The solved answer will be the hint for the succeeding question. Participant who solves maximum number of questions in minimum time will be qualified for the next round.
ROUND-2:
The participants will be provided with randomly arranged computers in which they are supposed to debug the given simple C code and the line number in which the error is present will be the hint for the next question. Participant who accomplishes the task in a minimum time frame successfully will be promoted to the next round.
ROUND-3:
A given C program has to be solved with accurate output. The participant who solves the program correctly first will be declared as the winner.
