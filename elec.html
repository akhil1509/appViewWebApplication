JUDGING CRITERIA : Based on the performance of an Individual.
DESCRIPTION
 ROUND-1:
The main objective of the contestant is to connect the source and load placed at two corners. That is, with the provided various pictures of insulators and conductors placed between the source and Load, the contestant have will to connect only the conductors but not insulators for conduction of circuit. Top 8 participants with maximum number of conductors placed will be qualified to next round.
ROUND-2:
The basic concept of this round is the classic game ’chess’. Each piece in a chess board is represented by an electrical component, for example: Soldier-resistor; Elephant-DC source; Camel-Inductor; Horse-Capacitor; Majesty-Current; King-AC Source Resistors, inductors, capacitors can move single step forward like a soldier Dc source can move in horizontal or vertical direction like an Elephant Current can move in horizontal, vertical or diagonal direction like a camel Ac source can move in sine shape like a horse ATTACKING SKILLS Resistor can kill only current (AC,DC) Dc source can kill inductor and capacitor Current can kill anyone Inductor can kill only Ac voltage Capacitor can kill only Ac current AC Source can kill anyone Resistor can kill anyone if opponent is diagonal
