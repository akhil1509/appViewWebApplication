Electro-Q




COST : 40/-
To register for the event contact Help desk at GRIET.
Spot registrations are available.
JUDGING CRITERIA : Based on the number of correctly answered questions in least time.
DESCRIPTION
 ROUND-1:
A cross word puzzle will be given to each team consisting of some electronic terms. The team has to search the name from the puzzle within a given time. The teams which answer them correctly in minimum time will be short listed to next round.
ROUND-2:
Electronic components will be given in a bowl. Participants need to find out the electrical components. The short listed candidates are qualified to next round.
ROUND-3:
In this both input and output waveforms are shown to the teams. They should name out the circuit name by observing the waveforms to which input is given.
